# Changelog

## 0.1.0

Initial working MVP:

- RViz `SensorTreePanel` plugin for ROS 2
- Automatic discovery for `LaserScan`, `Image`, and `PointCloud2`
- Device-tree grouping and configurable topic-grouping rules
- Automatic RViz display creation for discovered supported topics
- Demo publisher, local install scripts, RViz preset, and smoke test
